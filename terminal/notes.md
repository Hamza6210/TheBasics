# Day 1 - Terminal Basics

## What I learned
- Navigation: pwd, ls, ls -la, cd
- Creating files: touch, echo
- Reading files: cat, head, tail
- Writing vs appending: > overwrites, >> appends
- Copying, moving, renaming: cp, mv
- Deleting: rm (permanent, no trash bin)
- Searching inside files: grep, grep -i, grep -r
- Piping with | to chain commands together
- Counting lines with wc -l
